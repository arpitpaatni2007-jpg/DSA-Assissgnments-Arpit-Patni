# Sorting Performance Analyzer (SPA)

This project analyzes the performance of different sorting algorithms on various datasets.

## 📌 Features

* Insertion Sort
* Merge Sort
* Quick Sort (Random Pivot)
* Dataset generation (Random, Sorted, Reverse)
* Execution time measurement

## ▶️ How to Run

```bash
python spa_sorting.py
```

## 📊 Datasets Used

* Random data
* Sorted data
* Reverse sorted data
* Sizes: 1000, 5000, 10000

## 📈 Key Observations

* Insertion Sort is efficient for small or sorted data
* Merge Sort performs consistently across all datasets
* Quick Sort performs fast with proper pivot selection

## ⏱ Complexity

* Insertion Sort → O(n²)
* Merge Sort → O(n log n)
* Quick Sort → O(n log n) average

---

This project demonstrates how algorithm performance varies with input type and size.
