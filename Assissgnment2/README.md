# Linear Data Structures Toolkit (LDST)

This project focuses on implementing fundamental linear data structures using Python.

## 📌 Features

* Dynamic Array with resizing
* Singly Linked List
* Doubly Linked List
* Stack (LIFO)
* Queue (FIFO)
* Balanced Parentheses Checker

## ⚙️ Implementation Details

All data structures are implemented from scratch without using built-in methods.
Dynamic array resizing and linked list operations are handled manually.

## ▶️ How to Run

```bash
python ldst_toolkit.py
```

## 📊 Key Concepts

* Amortized complexity of dynamic arrays
* Efficient insertion and deletion using linked lists
* Stack and queue operations using linked list
* Expression validation using stack

## ⏱ Complexity Overview

* Dynamic Array Append → O(1) amortized
* Linked List Insert/Delete → O(1)
* Stack Operations → O(1)
* Queue Operations → O(1)
