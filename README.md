# DSA-Assissgnments-Arpit-Patni
# Data Structures and Algorithms Assignments

This repository contains implementations of core data structures and algorithms developed as part of the Data Structures and Algorithms course.

## 📂 Contents

* **Unit 2 – Linear Data Structures Toolkit (LDST)**

  * Dynamic Array
  * Singly & Doubly Linked List
  * Stack and Queue
  * Balanced Parentheses Checker

* **Unit 3 – Sorting Performance Analyzer (SPA)**

  * Insertion Sort
  * Merge Sort
  * Quick Sort
  * Performance comparison on different datasets

* **Unit 4 – Data Management Mini Toolkit (DMMT)**

  * Binary Search Tree (BST)
  * Graph (BFS & DFS)
  * Hash Table with Separate Chaining

## 🛠 Technologies Used

* Python 3
* Basic data structures implementation from scratch

## ▶️ How to Run

```bash
python filename.py
```
