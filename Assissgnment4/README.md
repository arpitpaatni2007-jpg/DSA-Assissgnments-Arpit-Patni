# Data Management Mini Toolkit (DMMT)

This project implements advanced data structures used in real-world systems.

## 📌 Features

* Binary Search Tree (BST)

  * Insert, Search, Delete, Inorder Traversal
* Graph (Adjacency List)

  * BFS and DFS Traversal
* Hash Table

  * Separate chaining for collision handling

## ▶️ How to Run

```bash
python dmmt_toolkit.py
```

## 📊 Key Concepts

* BST provides sorted data using inorder traversal
* BFS explores nodes level-wise
* DFS explores nodes depth-wise
* Hash tables provide fast key-value access

## ⏱ Complexity

* BST Operations → O(log n) average
* Graph Traversal → O(V + E)
* Hash Table → O(1) average
